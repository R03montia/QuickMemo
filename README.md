# QuickMemo

Win11 风格快捷备忘录 — 轻量、美观、本地优先。

## 特性

- **Win11 毛玻璃质感** — 透明窗口 + 模糊背景，跟随系统强调色
- **深浅主题** — 支持跟随系统、浅色、深色三种模式
- **灵活窗口** — 有边框/无边框自由切换，CSS 圆角裁剪
- **实时调节** — 内置不透明度滑块，随时调整面板透度
- **笔记管理** — 新建、编辑、拖拽排序、多选批量操作
- **提醒系统** — 自定义日期时间选择器，到点通知提醒
- **设置面板** — 左下角齿轮按钮，集中管理所有偏好
- **本地存储** — 数据保存在本地 JSON 文件，无需联网

## 快速开始

```bash
# 安装依赖
npm install

# 启动应用
npm start

# 开发模式（带 DevTools）
npm run dev
```

## 使用指南

### 笔记操作
- **新建笔记**：点击侧栏「新建笔记」按钮
- **编辑**：点击笔记列表中的笔记，右侧编辑区域会同步显示
- **重命名**：双击侧栏笔记标题直接内联编辑
- **删除**：点击编辑器底部删除按钮，或侧栏右键菜单删除
- **拖拽排序**：按住笔记左侧拖拽手柄上下拖动

### 多选模式
在笔记上右键 → 选择「多选」→ 勾选多个笔记 → 批量删除或设置提醒

### 提醒
点击编辑器中的「提醒」按钮 → 选择日期和时间 → 确定。到点系统会弹出通知。

### 主题与外观
系统托盘右键菜单可快速切换主题和窗口样式。左下角 ⚙ 齿轮打开设置面板进行更精细的调节。

### 窗口样式
- **有边框**：显示传统标题栏和窗口控制按钮
- **无边框**：隐藏标题栏，窗口控件浮动在右上角

## 技术栈

- **Electron** — 跨平台桌面框架
- **原生 HTML/CSS/JS** — 零前端依赖
- **contextBridge + ipcRenderer** — 安全的进程通信

## 数据存储

所有笔记和设置存储在 `data/notes.json` 文件中，纯 JSON 格式，方便备份和迁移。
